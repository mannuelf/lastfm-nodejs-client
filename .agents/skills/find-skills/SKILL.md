# Find Skills (Global Reference)

This is a local reference to the globally installed `find-skills` skill.

## Global Skill Location

The full skill is installed globally at: `~/.agents/skills/find-skills/SKILL.md`

## Usage

Use `npx skills find [query]` to discover and install agent skills from the open ecosystem.

### Key Commands

- `npx skills find [query]` — Search for skills interactively or by keyword
- `npx skills add <package>` — Install a skill from GitHub or other sources
- `npx skills check` — Check for skill updates
- `npx skills update` — Update all installed skills

### Browse Skills

Visit [skills.sh](https://skills.sh/) to explore available skills.
